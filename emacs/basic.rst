=====================
Basic 語法
=====================

指令
====================
*   Ctr = C
*   ESC = ESC


游標移動
====================

     指令
.. code-block:: sh
     C-p   	上
     C-n   	下
     C-b   	左
     C-f   	右
     ESC-b 	單字的頭
     ESC-f 	單字的尾
     C-a	行頭
     C-e  	行尾
     ESC- C-a	函式首
     ESC- C-e	函式尾
     ESC- <	檔案首
     ESC- >	檔案尾
     C-v  	下一頁
     ESC	上一頁


重複指令
=======================
   指令
.. code-block:: sh
   C-u		重複執行四次
   C-u C-u.....	重複執行四的n次方次
